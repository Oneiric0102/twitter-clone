# Twitter Clone

트위터 클론코딩 프로젝트

개발기간 : 2024.06 - 2024.08

## 배포주소

https://oneiric0102.github.io/twitter-clone/

## 주요기능

- 트윗 작성, 사진 첨부 가능
- 타임라인 보기
- 프로필 보기
- 팔로워 / 팔로우 목록 보기
- 프로필 사진 변경
- 유저 / 트윗 검색

## 사용 기술

- React
- TypeScript
- Firebase
- Styled-component
- React Hook Form
- React Dom Router
